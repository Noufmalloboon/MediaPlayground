
MediaPlayground App

An iOS playground app for experimenting with video and audio playback using AVPlayer and AVAudioEngine

Features:

- Video playback with custom AVPlayer controls
- Play / pause, seek, and playback rate control
- Live playback time updates
- Audio playback with custom AVAudioEngine graph
- Volume control and progress slider
- 3-band equalizer (Bass / Mid / Treble) with real-time effects
